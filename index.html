Tutorial: Post an article

Tutorial: Post an article Create a markdown file hugo new posts/file_name.md 将写好的.md文件内容放置在file_name.md文件内。 之后，需要添加文件的tags和categorise tags：一般是琐碎的，精准的 categories：一般是抽象的，泛指的 最后将draft设置为false，即可。 Preview the file 写好文章后第一步不应该commit，而是在本地查看是否符合预期。 hugo server 用上面的指令会先生成本地预览文件，在 http://localhost:1313/ 显示。 Commit 要提交article到网页，首先需要进入到 D:/hugo-papermod 的文件夹下 然后按照下面的指令操作即可。 hugo cd public git add . git commit -m "update" git push origin master Change favicon 首先需要生成favicon文件，到这个网站可以获取: https://www.logosc.cn/logo/favicon 然后将favicon文件全部放置在 D:/hugo-papermod/static 文件夹下面。然后转到 D:/hugo-papermod/config.yml 文件夹里面，修改为对应生成的favicon的文件即可
